# 视频文件存放说明

把你的视频文件改名后放进这个文件夹，网站就会自动识别并可以播放。
文件名必须和下表完全一致（区分大小写）。

| 文件名 | 对应网站上的作品 | 建议比例 |
|---|---|---|
| `work-01.mp4` | Aiper 泳池机器人海外广告混剪 | 16:9 横版 |
| `work-02.mp4` | Saylo AI 应用广告·中/英/日三语版本 | 9:16 竖版 |
| `work-03.mp4` | 小奥汀 Little Ondine 越南市场本地化 | 9:16 竖版 |
| `work-04.mp4` | 《小晨故事》重阳节专题短视频 | 16:9 横版 |
| `work-05.mp4` | "印象广州"短视频大赛获奖作品 | 9:16 竖版 |
| `work-06.mp4` | 大学生广告艺术大赛微电影广告 | 16:9 横版 |

放好之后，用浏览器打开根目录的 `index.html`，点击对应作品卡片上的
播放按钮 ▶️ 就能播放了。

## 没有视频文件时会怎样？

不影响网站其他部分正常浏览。点击播放按钮时会提示
"视频文件未找到"，并告诉你应该放哪个文件名 —— 这是正常的提示，
不是网站出错了。

## 想增加 / 减少作品数量？

打开根目录的 `index.html`，搜索 `work-grid`，里面每一个
`<article class="work-card">...</article>` 就是一个作品卡片，
复制一份改内容、改文件名（比如 `work-07.mp4`）即可新增；
删掉整段 `<article>...</article>` 即可减少。
