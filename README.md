# 徐玮婕 · 视频作品集网站

一个纯 HTML / CSS / JS 制作的单页作品集网站，无需任何安装、构建工具，
用浏览器直接打开 `index.html` 就能看效果。

```
portfolio-site/
├── index.html          ← 网站主文件（结构 + 样式 + 交互都在这一个文件里）
├── resume.pdf           ← 你的简历（已自动放入，点击"下载简历"按钮可下载）
├── assets/
│   └── profile.png      ← 头像照片（已自动从简历里提取）
└── videos/
    └── README.md         ← 视频文件命名说明（把你的视频按说明放进这个文件夹）
```

## 1. 怎么把你的视频放进去

网站「作品集」板块目前是 6 个占位作品卡片，对应了你简历里的真实项目
（Aiper 海外广告、Saylo 多语种素材、小奥汀本地化、娄底电视台新闻专题、
两个获奖作品）。

只需要：

1. 把你电脑里对应的视频文件改名为：
   `work-01.mp4`、`work-02.mp4` … `work-06.mp4`
2. 放进 `videos` 文件夹（具体哪个文件名对应哪个作品，见 `videos/README.md` 里的表格）
3. 用浏览器打开 `index.html`，点击对应作品卡片上的播放按钮 ▶️ 即可播放

> 如果你的作品和占位文案对不上（比如想换一支视频、改个标题），
> 直接打开 `index.html`，搜索对应的 `work-0X.mp4`，
> 把旁边的标题 `<h3>`、描述 `<p>` 文字改成你自己的内容即可，
> 不需要懂代码，普通的文字替换就行。

**视频建议规格**：MP4 格式（H.264 编码），单个文件建议不超过 50MB，
分辨率 1080p 即可，太大的文件加载会比较慢。

## 2. 怎么替换头像 / 简历

- 头像：把新照片改名为 `profile.png`（或 `profile.jpg`，并同步修改
  `index.html` 里 `<img src="assets/profile.png">` 这一行的文件名），
  放进 `assets` 文件夹覆盖即可
- 简历：把新的 PDF 改名为 `resume.pdf`，放在根目录覆盖即可，
  网站上"下载简历"按钮会自动指向这个文件

## 3. 本地预览

直接双击 `index.html` 用浏览器打开即可。

如果发现视频播放不了（部分浏览器对本地 `file://` 协议下的视频加载有限制），
可以用一个本地小型服务器预览，例如在这个文件夹下打开终端运行：

```bash
python3 -m http.server 8000
```

然后浏览器打开 `http://localhost:8000` 即可。

## 4. 免费部署上线（拿到一个可以分享的网址）

推荐两种零成本方式，几分钟就能搞定：

### 方式一：Vercel / Netlify（最简单，拖拽上传）
1. 注册一个 [vercel.com](https://vercel.com) 或 [netlify.com](https://netlify.com) 账号
2. 把整个 `portfolio-site` 文件夹直接拖到网站的部署区域（Netlify 首页有
   "Drag and drop your site folder here" 的区域）
3. 几十秒后会生成一个网址，比如 `xuweijie-portfolio.netlify.app`，
   投简历的时候直接附上这个链接即可

### 方式二：GitHub Pages（适合想顺便学一下 Git 的同学）
1. 在 GitHub 上新建一个仓库，把 `portfolio-site` 文件夹内容上传上去
2. 仓库 Settings → Pages → 选择分支并保存
3. 几分钟后会生成 `你的用户名.github.io/仓库名` 这样的网址

两种方式都不需要花钱，也不需要自己买域名。

## 5. 想自己改样式怎么办

所有样式都写在 `index.html` 顶部的 `<style>` 标签里，开头用 `:root{}`
定义了几个颜色变量，比如：

```css
--coral: #FF4B3E;   /* 主色：珊瑚红，对应"录制中"的红点 */
--yellow: #FFC83D;  /* 辅助色：时间码黄 */
--teal: #16A394;    /* 辅助色：声波绿 */
```

想换整体配色，改这几个变量的色值就行，其他地方都会跟着联动。
